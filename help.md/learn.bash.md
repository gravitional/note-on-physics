## ubuntu常用文件操作

+ `mkidr old` : 创建名为`old`的目录
+ `mv old new`: 将`old`目录命名为`new`
+ `rmdir old` : 目录删除分为删除空目录和非空目录，`rmdir`用于删除空目录
+ `rm -rf old` : 指令删除非空目录
+ `cp -r old /home` : 将`old`目录复制到`/home`路径下
+ `touch old` : 创建`old`文件，其他的过程都和目录的操作方法是一样的
